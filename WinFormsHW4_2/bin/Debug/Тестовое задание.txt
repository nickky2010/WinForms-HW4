Реализовать приложение на C# (Windows Forms/WPF), которое на входе принимает файл csv (текстовый файл, в котором поля разделены при помощи ';', 
формат файла см. ниже).
Приложение извлекает из него данные в базу данных. База данных - MS SQL Server. Доступ к БД реализовать при помощи Entity Framework. 
Приложение должно иметь функцию экспорта данных в Excel по любой комбинации полей (например, экспортировать данные за 01.01.2015 по городу Гомелю и фамилии Иванов).
Приложение должно иметь функцию экспорта данных в XML по любой комбинации полей.
Для выборки из базы данных использовать язык запросов LINQ. 

Исключить возможность ввода пользователем некорректных данных.
Реализовать обработку ошибок с выводом сообщений, понятных пользователю.
Пользовательский интерфейс приложения должен быть понятен даже неопытному пользователю.

Не допускается дублирование кода. 
Уделить внимание качеству кода, быстродействию, применению принципов ООП.


Формат файла csv:


Дата1;Имя1;Фамилия1;Отчество1;Город1;Страна1
Дата2;Имя2;Фамилия2;Отчество2;Город2;Страна2
Дата3;Имя3;Фамилия3;Отчество3;Город3;Страна3

Формат файла xml:

<TestProgram>
	<Record id="1">
		<Date>...</Date>
		<FirstName>...</FirstName>
		<LastName>...</LastName>
		<SurName>...</SurName>
		<City>...</City>
		<Country>...</Country>
	</Record>
	
	....
	
</TestProgram>
